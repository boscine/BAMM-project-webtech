# [MMA MMA]	#  
A web-based tool that lets students visualize the location of Wi-Fi
routers/access points across campus. When a student connects to campus
Wi-Fi and opens the app, the system identifies which router they're
connected to and highlights that router's position on an interactive
campus map.
